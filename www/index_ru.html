Простой переключатель раскладок клавиатуры.

Recaps - это переключатель клавиатуры для операционных систем семейства Windows, использующий для работы перенастраиваемую комбинацию клавиш либо одну клавишу, такую как используемая по умолчанию Caps Lock. Эта комбинация называется далее Переключателем (Cycle Switch).

Если у вас установлено более двух раскладок - вы можете организовать их в группы (Groups), активируемые и переключаемые своими собственными горячими клавишами (Group hotkeys). Для переключения внутри группы можно (и нужно!) использовать горячую клавишу Переключатель.

Если текст был случайно набран в неверной раскладке - нажмите горячую клавишу "Convert Inplace", чтобы автоматически исправить ошибку. Но успех операции порой сильно зависит от активной программы.

Если вы пользуетесь конфигурацией по умолчанию - с CapsLock в качестве Переключателя, оригинальная функциональность CapsLock доступна по горячей клавише "Left Alt + Caps Lock".

Индикатором текущей выбранной раскладки служит соответствуюший ей национальный флаг в области системных сообщений Windows, светодиод ScrollLock на клавиатуре либо полупрозрачный национальный флаг в центре экрана.

Отдельную горячую клавишу можно настроить и на запуск извлечения диска из компьютера. Эта возможность, к примеру, может быть использована с ноутбуками, не имеющими механической кнопки на CD/DVD приводе.

Если в пакете программы отсутствует национальный флаг для вашей раскладки - сообщите нам, и мы добавим его в следуюшие версии программы. Вы можете назначить такой раскладке любую иконку в формате 32 на 32 пикселя глубиной цвета True color (32 bit). Возможно использовать и другие форматы и размеры, но результат не всегда будет удовлетворительным.

Представленная выше примерная конфигурация имеет:
  • Группу из Немецкой и Грузинской раскладок, включаемую и переключаемую горячей клавишей "Left Control + 1";
  • Группу из Немецкой и Греческой раскладок, включаемую и переключаемую горячей клавишей "Left Control + 2";
  • Прямую горячую клавишу для Немецкой раскладки "Left Control + 3";
  • Прямую горячую клавишу для Грузинской раскладки "Left Control + 4";
  • После переключения на любую группу между её раскладками можно переключаться клавишей CapsLock;

Revisions (by Eli Golovinsky)

  • 0.6 Added conversion of text typed with the wrong layout using Ctrl-CapsLock. Alt-CapsLock now changes the old CapsLock mode.
  • 0.5 Fixed selected languages configuration not being saved in the registry.
  • 0.4 Fixed language not being switched in some applications.
  • 0.3 Added a tray icon, a menu and support for more than two languages. Recaps now cycles between the lanugages selected in the menu.
  • 0.2 Changed Shift-CapsLock to change language as well as CapsLock. Ctrl-CapsLock now changes the old CapsLock mode.
  • 0.1 Initial release.

История изменений:

Версия 0.9.00.3850  

  • Полное название региона, языка и клавиатурной раскладки в настройках.
  • Восстановлена функция изменения ширины колонок в списках.
  • Исправлено закрытие диалога настроек экранной подсказки по Esc.
  • Полные имена языков в колонках настройки групп.
  • Исправлено выравнивание заголовков колонок.
  • Горизонтальный размер диалога настроек сделан изменяемым.
  • Исправлена отрисовкa слишком длинного текста в ячейках списков.
  • Окна учитывают системные настройки Windows для корректного отображения увеличеного шрифта.
  • Добавлены некоторые национальные флаги. На основании статистики скачиваний программы.

Версия 0.9.00.3676  

  • Добавлена рудиментарная справка об активных областях диалога настройки;
  • Предотвращается открытие копий дубликатов диалога настроек;
  • В диалоге конфигурации используются имена раскладок вместо имён языков;
  • Авто-конфигурация горячея клавиши переключателя CapsLock учитывает группы и ставится в Alt-CapsLock если группа задействует CapsLock для себя;
  • Исправлено отключение горячей клавиши циклического переключения. Она сохраняет своё состояние после перезапуска программы;
  • Улучшено запоминание текущей выбранной группы. Приложение стартует с последней выбранной группой;
  • Поддержка экранного индикатора раскладки на мульти-мониторных системах;
  • Реализована гибкая конфигурация затухания экранного индикатора с национальным флагом;
  • Предотвращено автоматическое закрытие диалога конфигурации горячей клавиши если эта клавиша содержит код пробела;
  • Расширенный набор иконок национальных флагов, созданный на основе статистики скачиваний Recaps;

Версия 0.9.00.3480  

  • Настройка через контекстное меню заменена конфигурационным диалогом;
  • Поддержка горячих клавиш с модификаторами - такими как Control, Shift, Alt и т.д.;
  • Добавлена возможность объединять раскладки в группы. Каждая группа имеет свою горячую клавишу, а также переключается основным Переключателем;
  • Дополнительная возможность отображения текущей раскладки - полупрозрачный национальный флаг в середине экрана;
  • Использован более качественный формат иконок национальных флагов 32 x 32 пикселя True color. Добавлены новые национальные флаги;
  • Монитор ключа HKCU\Keyboard Layout\Preload для автоматической синхронизаци изменений в списке раскладок;
  • В установочный пакет добавлена возможность установки для пользователей не имеющих прав администратора;

Версия 0.9.00.400  

  • Исправлено аварийное завершение при конвертировании набранного текста по Ctrl-[Клавиша];

Версия 0.9.00.376  

  • Поддержка Windows x64;
  • Нaстраиваемый пользователем значок в области системных сообщений Windows;
